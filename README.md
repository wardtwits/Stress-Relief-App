# Stress Relief Simulator Website

This is a minimal static website for promoting Stress Relief Simulator.

## Files

- `index.html` — home page with app summary and screenshots
- `privacy.html` — privacy policy
- `css/style.css` — shared styles
- `assets/` — local screenshots used by the site

## View locally

Open `index.html` directly in a browser, or serve the `website/` folder with any static file server.

Examples:

- `cd website && python3 -m http.server`
- then open `http://localhost:8000`

## Update the privacy policy

If the app’s practices change, update `privacy.html` to reflect:

- whether the app collects any personal or device data
- whether it uses analytics, ads, accounts, or third-party SDKs
- whether any data is transmitted off-device
- the effective update date near the top of the page
- the contact email if it changes

Keep the policy aligned with the real app behavior.
